  

nearest_neighbors.py contains our implementation of Nearest Neighbors. Please make sure you have the dataset in the same folder you are calling the function.  The program can be run with the command "python nearest_neighbors.py" The program prints out the Accuracy, precision, recall and F value for all files given in array in main method on line 132. The value of k can be changed on line 133.

naive_bayes.py contains our implementation of Naive Bayes. The program can be run with the command "python naive_bayes.py." The program prints out the Accuracy, precision, recall and F value for all files given in array in main method on line 199. The query can be changed on line 192
